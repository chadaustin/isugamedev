wpaint1.exe

This program satisfied (almost) every assignment requirement. It implemented rubber banding by remembering every command that had been drawn, and therefore could remove the most recently-created command and add the new one.

wpaint2.exe

I added transparency to this version. The new "Transparency" menu allows you to select any translucency level between 0% and 100% in 10% increments.

wpaint3.exe

Version 3 has support for undo and redo. This was trivial to implement.
