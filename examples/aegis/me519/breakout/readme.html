Chad Austin's Breakout

Since I've been really busy, I couldn't spend as much time as I would have liked on this project. It has a few bugs in the collision detection routines... Sometimes the ball may get stuck in the walls or travel up through the crack between two blocks. I did add sound effects using my audio library, however (and thereby found a few issues with the performance of the library in interactive situations).

I think the assignment took me three hours.

FileDescription
breakout.zip Win32 binaries, including Audiere DLL and sound files
breakout.cpp Main GLUT interface, game state machine
color.hpp Simple color struct
game_state.{c,h}pp Implementation of State interface, actual Breakout logic
null_state.{c,h}pp Implementation of a state which does nothing
point.hpp Simple point struct, related operations
rectangle.hpp Simple rectangle struct
sound.{c,h}pp Sound effect engine
state.hpp Definition of State interface
string_state.{c,h}pp State implementation that draws a string to the screen
text.{c,h}pp Utility functions to draw text using GLUT fonts
tuple.hpp Both vectors and points are tuples
vector.hpp Simple vector struct, related operations
