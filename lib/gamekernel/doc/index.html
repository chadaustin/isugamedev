Introduction

GameKernel is a flexible, open source platform for running games being developed by the ISU Game Developers Club. It is developed under the LGPL and can be used freely. It runs on Windows, Linux and Irix.

Your game should not depend on the underlying system API. You should be able to write your game and know that it will immediately port to any system without having to modify more than a single line of code. GameKernel achieves this portability. Your game can be running under GLUT, SDL, VRJuggler or any other of a myriad of system abstractions. Your application will never notice the difference and will work beautifully the first time.
