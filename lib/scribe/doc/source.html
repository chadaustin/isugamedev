Source Code

If you want the latest, probably broken, source to Scribe, you can get it from the CVS repository on SourceForge.net.

Anonymous CVS Access

Scribe's SourceForge CVS repository can be checked out through anonymous (pserver) CVS by using the following instructions. When prompted for a password for anonymous, simply press the Enter key. 
cvs -d:pserver:anonymous@cvs.isugamedev.sourceforge.net:/cvsroot/isugamedev login

cvs -z3 -d:pserver:anonymous@cvs.isugamedev.sourceforge.net:/cvsroot/isugamedev co lib/scribe

Developer CVS Access via SSH

Only developers of the isugamedev SourceForge project can access CVS via this method. SSH1 must be installed on your client machine. Enter your SourceForge site password when prompted. The following instructions are based on a sh-style shell. Substitute yourname with your SourceForge username. 
export CVS_RSH=ssh

cvs -z3 -d:ext:yourname@cvs.isugamedev.sourceforge.net:/cvsroot/isugamedev co lib/scribe
