Introduction

Scribe is a flexible, open source scripting engine being developed by the ISU Game Developers Club. It is developed under the LGPL and can be used freely. It runs on Windows, Linux and Irix.

Scribe is not intended to be a replacement for COM or XPCOM. Instead Scribe attempts to be a lightweight language bridge.

This project is driven by the idea that a game engine shouldn't have to change (much) if some of the logic is moved to a different language. In addition, if a new language is added to the scripting engine, the interface should be unchanged.
